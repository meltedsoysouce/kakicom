# Kakicom PKM

思考の断片を無限キャンバス上に配置し、関係・変化・不足を可視化する**認知拡張ツール**。

## セットアップ

```bash
cd app
pnpm install
pnpm dev        # http://localhost:5173
```

## 操作方法

### キャンバス

| 操作 | 説明 |
|------|------|
| ドラッグ（背景） | パン（キャンバスの移動） |
| マウスホイール | ズーム（カーソル位置を中心） |

### Node

| 操作 | 説明 |
|------|------|
| 背景をダブルクリック | 新規Nodeを作成し、テキスト編集を開始 |
| Nodeをクリック | Nodeを選択 |
| Nodeをダブルクリック | テキスト編集モードに入る |
| Nodeをドラッグ | Nodeを移動 |

### Edge（Node間接続）

| 操作 | 説明 |
|------|------|
| `L` キー | リンクモードの切替（カーソルが十字に変化） |
| リンクモード中にNodeをクリック | 1回目: ソースNode選択、2回目: ターゲットNode選択 → Edge作成 |
| Edgeをクリック | Edgeを選択 |
| `Delete` / `Backspace` | 選択中のEdgeを削除 |
| `Escape` | リンクモードのキャンセル / 選択解除 |

### ファイル保存・読込

| 操作 | 説明 |
|------|------|
| `Ctrl+S` (`Cmd+S`) | エクスポート — 全Node・Edgeを `.kakicom.json` ファイルに保存 |
| `Ctrl+O` (`Cmd+O`) | インポート — `.kakicom.json` ファイルを読み込み（既存データを全置換） |

### その他

| 操作 | 説明 |
|------|------|
| `Escape` | 選択を解除 |

## 技術スタック

- TypeScript 5.9 (strict mode)
- Vite 7 (rolldown-vite)
- Canvas 2D
- IndexedDB (ブラウザローカル永続化)
- UIフレームワークなし
